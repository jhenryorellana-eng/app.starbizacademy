# Starbiz Academy - Hub Central

Portal de autenticación y gestión familiar para Starbiz Academy.

## Tech Stack

- Next.js 15 + React 19
- Supabase (Auth + DB)
- Stripe (Pagos)
- Tailwind CSS

## Desarrollo Local

```bash
npm install
npm run dev
```

## Despliegue

Este proyecto está configurado para desplegarse en Vercel. Las variables de entorno deben configurarse manualmente en el dashboard de Vercel.
